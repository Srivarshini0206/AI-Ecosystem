# AI Ecosystem – Simple Prompt Formatter

## Overview

AI Ecosystem is a beginner-friendly Python project designed to help users understand the basics of prompt engineering, AI applications, and chatbot structure. This project allows users to create, format, edit, save, and load structured prompts through a simple text-based menu system.

## Features

* Create new AI prompts
* Format prompts with task, style, constraints, examples, and output type
* Save prompts for future use
* Load existing prompts
* Edit prompts easily
* Beginner-friendly command-line interface

## Technologies Used

* Python
* File Handling
* Modular Programming
* Command Line Interface (CLI)
